## 显示计划

显示当前会话中正在执行或已执行的计划。

### 使用方法

```bash
/show-plan
```

### 基本示例

```bash
# 查看当前计划
/show-plan
“显示正在执行的计划”

# 没有计划时
/show-plan
“当前会话没有计划”
```

### 功能

- 检测由 `exit_plan_mode` 创建的计划
- 搜索包含“实施计划”、“实施内容”、“计划”等关键字的标题
- 格式化并显示计划内容
- 如果没有计划，则明确通知

### 与 Claude 协作

```bash
# 在实施过程中查看计划
“我正在实施什么来着？”
/show-plan

# 正在执行多个任务时
“让我再确认一下现在的计划”
/show-plan

# 计划执行后回顾
“让我看看刚才执行的计划内容”
/show-plan
```

### 检测模式

根据 `exit_plan_mode` 生成的计划格式，检测以下模式：

- 以 `##` 开头的标题（包含“计划”、“プラン”、“Plan”）
- `### 变更内容`
- `### 实施内容`
- `### 实施计划`
- `### 1.` 等编号标题

### 注意事项

- 仅显示当前会话中的计划（不包括过去的会话）
- 优先显示最新的计划
